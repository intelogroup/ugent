Architecture: Medical Textbook RAG

Fast, Accurate, and Multimodal Retrieval

1. Data Processing Pipeline

Text Chunking

Recursive character splitting with overlap. Context-aware headers.

→

Metadata Tagging

Book, Chapter, Page, Topic, Section Headers.

→

Pinecone Index

Serverless Index (text-embedding-3-large, 3072 dims).

2. Retrieval Strategy

Semantic Search

  • Uses OpenAI Large embeddings
  • Captures conceptual meaning
  • Excellent for natural language queries

Keyword/Metadata Filters

  • Ensures specific book/chapter focus
  • Handles medical jargon/abbreviations
  • Strict filtering for accuracy

3. Key Optimization: Topic-Aware Context

Instead of just text fragments, we inject the Breadcrumb Metadata (e.g., Biochemistry > Molecular > Nucleotides) into the embedding content to ensure the AI "understands" the context of every snippet.
