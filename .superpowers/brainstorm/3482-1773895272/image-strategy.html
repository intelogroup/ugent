Image Integration Strategy

How should we index the visuals from First Aid and Pathoma?

A

Text-Linked (Simple & Fast)

Index the captions and surrounding text. When you search for "Mitral Stenosis", the system finds the text and pulls up the associated diagram/echo image based on proximity.

  • Uses existing text-embedding-3-large pipeline
  • Reliable and easy to implement
  • Lower cost
B

Multimodal (Advanced & Accurate)

Use a Vision-enabled model (like CLIP or GPT-4o) to "describe" every image into text, or use multimodal embeddings. You can search for visual patterns even if the caption is vague.

  • Can find images based on visual content
  • Requires an extra processing step per image
  • Highest search accuracy for diagrams/slides
